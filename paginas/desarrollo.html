(DEFUN C:ELF () (alert "Comando ELF (EXPORTA LINEAS FRENTISTAS)") (setvar "cmddia" 0) (setq unidad "D:\\") (setq archivo (open (STRCAT unidad"CARGA\\lista.TXT") "r")) (setq numlinea (read-line archivo)) (while (/= numlinea "FIN") (setq sec numlinea) (setq path (strcat unidad"SECCIONES\\" sec ".DWG")) (COMMAND "ADEDRAWINGS" "ATTACH" path "x") (COMMAND "ADEQUERY" "CLEAR" "DEFINE" "PROPERTIES" "LAYER" "PARCELAS_SIN_PROCESAR" "DATA" "AND" "OBJDATA" "LADO_PARCELA" "SECCION" "=" sec "DATA" "ANDNOT" "OBJDATA" "LADO_PARCELA" "LINDERO" "=" "PARCELA" "DATA" "ANDNOT" "OBJDATA" "LADO_PARCELA" "OCHAVA" "=" "X" "X" "E" "D") (COMMAND "ADEDRAWINGS" "DETACH" "*" "EXIT") (COMMAND "ZOOM" "E") (command "-mapexport" "shp" (strcat unidad "SECCIONES\\SHP\\" sec "fre") "yes" "R:\\SECCIONES\\AUXILIARES\\exportar_lineas_parcela_frentes.epf" "Proceed") (command "erase" "all" "") (setq numlinea (read-line archivo)) ) (CLOSE archivo) (setvar "cmddia" 1) )

Al cargar la rutina en AutoCAD Map mediante la línea de comandos con APpload e ingresando el comando ELF, el sistema consultará un archivo de texto con los datos requeridos. Luego, mediante ADEQUERY, extraerá la información desde un archivo DWG y la representará gráficamente en el entorno de AutoCAD. Finalmente, los datos procesados se exportarán como un archivo SHAPEFILE, almacenado en una ruta específica.
